# 🎲 Fun Fact Generator

A simple and interactive web app that generates random fun facts with a single click. Built as a beginner-friendly project to explore HTML,CSS,Javascript and github

## Deployment link : https://fun-fact-generator-lemon.vercel.app/
---

## 🚀 Features

* 🎯 Generates random fun facts instantly
* ⚡ Fast and lightweight
* 🖱️ Easy-to-use interface

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

---

## 📂 How It Works

* A predefined list (array) of fun facts is stored in JavaScript
* On button click, a random index is generated
* The corresponding fact is displayed on the screen
